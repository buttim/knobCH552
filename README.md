KnobCH552
===
USB volume knob based on CH552 board, AS5600 magnetic rotary sensor and WS2812 16 LEDs ring (optional).

Connections
---
- WS2812 data line on P3.3
- WS2812 V+ on 5V
- AS5600 SCL on P3.5
- AS5600 SDA on P3.4
- AS5600 V+ on 3.3V
- AS5600 DIR on GND
- touch sensors on P1.5, P1.6 and P.17
